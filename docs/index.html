Option-Implied Geopolitical Risk

Investment Hedging Model — Impacts of Geopolitical Tensions on Cross-Asset Allocations

Tracking how the Iran–Israel–U.S. conflict and Strait of Hormuz disruptions moved oil, airline, consumer, healthcare, and broad-market risk — and testing whether Black-Scholes-based hedges would have actually protected a client portfolio through it.
Built across seven notebook phases: Black-Scholes engine → market data pipeline → implied volatility → volatility surfaces → geopolitical event study → portfolio hedging → model-risk analysis. Phases 5–7 (this dashboard) run on real historical price data and VIX/OVX volatility proxies. Phases 2–4 (pricing-engine validation, market-data pipeline, implied-vol surfaces) used synthetic data generated from the same Black-Scholes engine, since historical per-strike option chains require paid data access this project didn't have. That substitution is intentional and documented throughout, not hidden.
